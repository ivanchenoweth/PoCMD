Sistema de logueo "SimpleLoginSystemMasterDetail"
Incluye:
 - CRUD de Usuarios
 - CRUD de Productos
 - CRUD de Clientes
 - CRUD de Proveedores
 - Simple ACL: Usuarios (Admin, RegistredUser, Guest)
 - Master-Detail Venta 
 - Sistema Simple de logueo using sesiones.
 - Prueba de unidad: "unitTestModeloUsuarios.php" usando phpunit framework.
 - No contiene diseño gráfico, ni validación de datos.
 - Tecnologías: Solo uso de: PHP-MYSQL-HTML-UML
 
 Todas las clases han sido generadas desde ArgoUML atraves de los diagramas de clases, para exportarse como esqueleto y sincronizados con el código. 
 Significa que, exportando el código de las clases, se puede continuar desarrollando dentro de los métodos.
 Es decir, se está usando ingeniería de software directa, donde es portante respetar las directivas de ArgoUML directivas (begin-end) en los métodos. 
 
 INSTALACION/CONFIGURACION:
 Para instalarse solo se necesita:
1.- Colocar los scrpits PHP "SimpleLoginSystemMasterDetail" disonibles en el servidor HTTP 
2.- Crear una base de datos "loginsystem" en mysql.
3.- Ejecutar el archivo "loginsystem.sql" en el servidor SQL.
4.- Ingresar al sistema por el navegador
5.- Para ingresar los usuarios predeterminados son:
